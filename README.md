# AREA 51 RAID SQUAD

### To start locally

1) Clone the repo
2) cd into the repo
3) ```npm install```
4) ```npm start```
5) Open your browser to http://localhost:3000

### To run in a docker container and tag

1) ```docker-compose up -d --build```
2) Open your browser to http://localhost:3001
